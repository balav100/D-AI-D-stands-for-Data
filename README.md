# D-AI-D-stands-for-Data
Project

Introduction:-
D-AI is a local AI-powered data analysis application that lets users explore datasets, generate insights, and interact using a chatbot and voice assistant — all running offline using Llama3 (Ollama).

Features:-
Upload CSV / Excel datasets
Data preprocessing
Handle missing values
Remove duplicates
Interactive visualizations
AI chatbot for dataset-based questions
Voice assistant (speech → AI → audio response)
Anomaly detection (Isolation Forest)
What-if simulation
PDF report generation
User authentication with Firebase

Tech Stack:-
Frontend: Streamlit UI
Backend: Python
AI Model: Ollama[Local LLM]
Visualization: Matplotlib, Seaborn
Voice Processing: SpeechRecognition, gTTS
Real Time Database & Authentication: Firebase
